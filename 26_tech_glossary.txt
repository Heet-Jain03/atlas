TECH GLOSSARY FOR BEGINNERS
=============================

API (Application Programming Interface)
  A way for two software programs to talk to each other. Like a waiter
  taking your order to the kitchen and bringing back your food.

Cloud Computing
  Storing and accessing data/programs over the internet instead of on
  your local computer. Examples: Google Drive, iCloud, Dropbox.

CPU (Central Processing Unit)
  The "brain" of a computer that carries out instructions.

DNS (Domain Name System)
  Translates website names (like google.com) into IP addresses computers
  can understand. Think of it as the internet's phone book.

Encryption
  Scrambling data so only authorized people can read it.

Firewall
  A security system that monitors and controls incoming/outgoing network
  traffic based on rules.

HTTP / HTTPS
  Protocols for transferring web pages. HTTPS is the secure (encrypted)
  version. Always prefer HTTPS.

RAM (Random Access Memory)
  Short-term memory your computer uses while running programs. More RAM
  = better multitasking.

VPN (Virtual Private Network)
  Encrypts your internet connection and hides your real IP address.
  Used for privacy and secure remote access.
